# 🍋 Little Lemon Restaurant

This is the final Capstone Project for the **Meta Front-End Developer Professional Certificate** on Coursera. Little Lemon is a family-owned Mediterranean restaurant, focused on traditional recipes served with a modern twist.

## 🚀 Features
* **Table Reservation:** A fully functional booking form with validation.
* **Responsive Design:** Optimized for mobile, tablet, and desktop views.
* **Modern UI/UX:** Designed using Figma and implemented with React.
* **Unit Testing:** Form validation and logic tested using Jest.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** CSS3 / SASS
* **Testing:** Jest & React Testing Library
* **Design:** Figma

## 🎯 Project Goals
The main objective was to apply all the skills learned throughout the Meta certification program, including UX/UI principles, React component architecture, and robust testing.
